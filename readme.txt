== Description ==

Push notification wordpress allows administrator to show simple notifications to specific users or to all users on their site. Administrator can also customize the appearance of each notification.

<strong>Key features:</strong>
<ul>
<li>Add notification for specific users or for all site users</li>
<li>Customize appearance of each and every notification individually</li>
<li>Add unlimited notifications</li>
</ul>

== Installation ==

1. Upload `push-notification-wordpress.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPresss

== Frequently Asked Questions ==

= Can I change appearance of each notification separately? = 

Yes! You have all that freedom!

= How can I specify which notification to show to which user? =

You can select users while creating or editing a notification. All the users mentioned there will be shown the notification.
